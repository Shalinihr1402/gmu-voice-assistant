GMU VoiceBot Assistant 🎙️🤖
GMU VoiceBot Assistant is an AI-powered multilingual university voice assistant designed for Indian students.
It supports voice conversations in English, Hindi, and Kannada, helping students access university information naturally using speech.
The project integrates:
🎤 Vapi AI Voice Assistant
🧠 OpenAI Intelligence
🔊 Real-time Voice Conversations
🗄️ MySQL Database
🌐 React Frontend
⚙️ PHP Backend APIs

⚙️ Installation
1️⃣ Clone Repository
Bash
git clone https://github.com/shalini1402/gmu-voice-assistant.git
2️⃣ Navigate to Project Folder
Bash
cd gmu-voice-assistant

📦 Frontend Setup
Bash
cd frontend
npm install
npm run dev
⚙️ Backend Setup
Place the project inside the XAMPP htdocs folder.
Start:
Apache Server
MySQL Server
Backend URL:
Bash
http://localhost:8080/gmu-voice-assistant/backend/
🗄️ Database Setup
Create MySQL database:
SQL
CREATE DATABASE gmu_voice_assistant;
Import your SQL database file.
Configure database connection in:
Bash
backend/config/database.php
